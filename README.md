# dAI 

Sistem za analizo delovnih navad in učinkovitosti dela za računalnikom.

## Tehnologije

- **STM32F411** — zajem IMU podatkov (giroskop, akcelometer, magnetometer)
- **Python 3.11+** — parsanje in vizualizacija
- **NumPy** — obdelava podatkov
- **Matplotlib** — vizualizacija
- **pyserial** — serijska komunikacija
